# ☕ Coffee Application

Bu proje, ASP.NET Core MVC kullanılarak geliştirilmiş bir **kahve menüsü ve blog uygulamasıdır**.  
Kullanıcılar menüdeki sıcak, soğuk içecekler ve tatlılar arasında gezinebilir, detayları inceleyebilir ve kendi kahve tasarımlarını oluşturabilir.  

---

## 🚀 Özellikler

- Kahve menüsü (Sıcak / Soğuk / Tatlı kategorileri)  
- Üye girişi ve sadece üyelerin erişebildiği **kahve tasarla** sayfası  
- Blog sayfası (kahve kültürü ve paylaşımlar için)  
- Responsive tasarım (Bootstrap)  
- SQL Server veritabanı entegrasyonu  

---

## 🛠 Kullanılan Teknolojiler

- **ASP.NET Core MVC**  
- **Entity Framework Core**  
- **SQL Server**  
- **Bootstrap 5**  
- **C#**  

---

## 📂 Proje Yapısı

- `Controllers/` → MVC Controller sınıfları  
- `Models/` → Veritabanı modelleri  
- `Views/` → Razor View dosyaları  
- `wwwroot/` → CSS, JS ve görseller  

---

## 🔧 Kurulum

1. Projeyi bilgisayarınıza indirin veya klonlayın.  
2. `appsettings.json` dosyasındaki **ConnectionString** kısmını kendi SQL Server ayarlarınıza göre güncelleyin.  
3. Visual Studio üzerinden `Update-Database` komutu ile veritabanını oluşturun.  
4. Uygulamayı çalıştırın 🚀  

---
